# WikiPlag - Plattform zur Erkennung von Wikpediaplagiaten

## Übersicht

Projektstudiumsaufgabe im SS16 an der HTW Berlin, betreut von Prof. Dr.-Ing. Hendrik Gärtner

##Build
```
sbt package assembly
```
Goal: package --> Webapp

Goal: assembly --> Backend fat-jar
